# Camera Recorder App

## Описание
Приложение для записи видео с камеры Teruhal Z2 и загрузки на Google Диск.

## Технологии
- React Native (Expo)
- Firebase Realtime Database
- Google Drive API (Service Account)

## Структура Firebase
- `/recordings` - метаданные записей
- `/cameras` - настройки камер

## Сборка
Запустить GitHub Actions → Скачать APK

## Ветки
- `main` - основная
- `backup-*` - бэкапы
